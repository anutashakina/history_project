# Спрятанный город

Цифровой исторический атлас о спасении архитектурного наследия блокадного Ленинграда.

## О проекте

Сайт посвящён памятникам и архитектурным объектам Ленинграда, которые в годы блокады маскировали, укрепляли и укрывали от обстрелов. Материал построен как последовательный просмотр: при прокрутке страницы активируется очередной объект, а его положение отмечается на карте.

## Технологии

- React
- TypeScript
- Vite
- Tailwind CSS
- Motion

## Запуск

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
```

## Структура

- `src/app/components` — секции сайта и интерфейсные компоненты
- `src/app/data/monuments.ts` — данные объектов
- `src/styles` — шрифты и глобальные стили
- `.github/workflows/deploy.yml` — автоматическая публикация
